Клиентская Веб-разработка (HTML, CSS, JavaScript)

Старт: идет набор

Цена: 12000 руб. 13000 руб. (возможность оплаты по частям)

Цена для студентов: 9000 руб.

Содержание курса:

В курсе веб-разработки мы познакомим вас с киентской веб-разработкой, которая включет в себя верстку - обучение HTML и CSS, и написание кода на JavaScript. На курсе JavaScript будут рассмотрены синтаксис, типы данных, работа с DOM и событиями, стандартная библиотека языка.

В курсе JavaScript также рассматривается большое количество библиотек и средств разработки - jQuery, jQuery UI, Underscore, Vue.js, Bootstrap, npm, Webpack, Less/Sass, Babel. Так же обзорно рассматриваем Grunt, RequireJS, TypeScript. В качетсве серверной части используем Express.js. Обучение веб-разработке закрепляется на практике.

Вы научитесь создавать клиентскую часть сайта и получите большой кругозор по клиенстким технологиям.
